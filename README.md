# Dash Ride Noir

Dash Ride Noir is a ride-sharing data analysis dashboard built with a React frontend and a Flask backend. It provides insights into trips, users, drivers, and payments through interactive visualizations.

## Overview

The project focuses on analyzing ride-sharing data and presenting it through a web-based dashboard. The backend handles data processing and APIs, while the frontend displays charts and metrics.

## Tech Stack

### Frontend
- React
- TypeScript
- Tailwind CSS
- Recharts
- Lucide React

### Backend
- Flask
- SQLite

## Features

- Dashboard with key metrics
- Trip analysis and visualization
- Driver and rider data insights
- Payment and revenue tracking
- REST API using Flask
